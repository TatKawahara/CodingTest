# 本リポジトリについて
コーディングテスト「最長片道きっぷの旅」のソースコードや解法をまとめたものです。

# 各ファイルについて
- `solution.cpp` には問題を解くための C++ のソースコードが書かれています。
- `document.md` には解法についての説明が書かれています。

# プログラムの実行について
- `solution.cpp` に書かれているコードをそのままコンパイル、実行していただくと動くと思います。依存関係にあるファイルなどはございません。
- プログラムの実行時間制限は 2 秒に設定しています。変更する際は、プログラム 232 行目の `kTimeLimit` (単位はミリ秒です) の値を書き換えてください。
